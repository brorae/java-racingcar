# StringCalculator
1. 커스텀구분자를 확인하는 기능
   * 기본구분자는 ',',':'이다.
   * 커스텀구분자일 경우, 기본구분자에 더해준다.
2. 구분자를 기준으로 문자열을 나눠주는 기능
3. 문자열의 숫자를 더해주는 기능
   * [예외] 문자열이 숫자가 아닌경우, RuntimeException을 발생시킨다.
   * [예외] 문자열이 음수인 경우, RuntimeException을 발생시킨다.
   * [예외] 문자열이 공백인 경우, 0을 리턴한다.

   
자동차 경주 미션 저장소
